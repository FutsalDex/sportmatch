# SportMatch AI: Elite Digital Scouting Terminal

**SportMatch AI** es una plataforma SaaS de alto rendimiento diseñada para digitalizar y optimizar el ecosistema del fútbol profesional y semi-profesional (Fútbol 11 y Fútbol Sala). Conecta de forma inteligente a Jugadores, Entrenadores y Clubes mediante un sistema de scoring impulsado por IA.

## 🚀 Misión del Proyecto
Transformar el scouting tradicional en una experiencia de datos y multimedia, eliminando las barreras geográficas y facilitando la contratación técnica mediante expedientes contractuales transparentes y detallados.

## 🛠️ Stack Tecnológico
- **Frontend**: Next.js 15 (App Router), React 19.
- **Estética**: Tailwind CSS + ShadCN UI (Diseño "Dark Mode" cinematográfico y futurista).
- **Backend**: Firebase (Firestore, Auth, Storage).
- **Inteligencia Artificial**: Genkit + Google Gemini (Análisis de perfiles y resúmenes automáticos).

## 💎 Pilares Funcionales

### 1. Identidad Deportiva de Élite
- **Perfiles Multidimensionales**: Biografías técnicas, historial de carrera por temporadas y logros.
- **Visual Scouting**: Integración de vídeo (YouTube) y redes sociales (Instagram/TikTok) directamente en el perfil.
- **AI Score**: Un índice de relevancia (0-100) calculado dinámicamente basado en la completitud y calidad del perfil.

### 2. Terminal de Reclutamiento (Mercado de Fichajes)
- **Tablero de Ofertas Público**: Un hub centralizado donde los clubes publican vacantes activas.
- **Ingeniería Contractual**: Las ofertas incluyen parámetros avanzados de Competición, Categoría, Salarios y Logística.

### 3. Sistema de Suscripción (SaaS Model)
- **Planes Pro y Top**: Acceso a análisis de IA avanzado, informes PDF y mayor visibilidad para el scouting profesional.
